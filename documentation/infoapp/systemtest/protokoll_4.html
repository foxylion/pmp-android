Tobias Kuhn

Die Einschätzung "Nicht Bestanden" bei NFT-5 ergibt sich aus der Kombination mehrerer Einschätzungen:

  • Die Info-App selbst wirkt solide kommentiert, die Qualität der einzelnen Kommentare und des Codes lässt teilweise zu wünschen übrig. Der Code verstößt gegen die PMP-Formatter/Cleanup-Style.
  • Die Info-App-Communication-Lib scheint vorbildlich kommentiert.
  • Dem Info-App-Filler-Projekt fehlen Kopfkommentare, insbesondere Autorenangaben. Die Einbindung von JSON.org-Bibliotheken als Code ohne entsprechende Dokumentation warum erscheint fragwürdig. Im Code sind Zeilen einfach auskommentiert (z.B. eine tote Methode in der Main-Klasse) oder vorgenerierte TODO-Kommentare enthalten.
  • Das Info-App-Pager-Indicator-Projekt ist eher spartanisch kommentiert und bindet bei Deklaration und Benutzung undokumentierten Code ein, der mit dem Copyright von Drittparteien versehen ist.
  • Die Connection-RG wirkt ausreichend dokumentiert, allerdings qualitativ eher mittelmäßig.
  • Die Energy-RG wirkt unzureichend dokumentiert. Auffällig waren neben generell fehlenden (Kopf-)Kommentaren auch Zeilen wie "// TODO: Clear database" oder undokumentierte, leere Catch-Blöcke. Der Code verstößt gegen die PMP-Formatter/Cleanup-Style.
Insgesamt führt dies zu einem insbesondere lizenzrechtlich unzureichenden Eindruck, obwohl eine breite Palette an Qualitäten vorhanden ist.
